CMDIClientView::MinimizeView
Minimizes a view
virtual void MinimizeView(CMDIView *view);


Remarks
This method is called when a MDI_MSG_MINIMIZE_VIEW message is received, but it can also be called directly. This implementation simply hides the view, but a derived class could use overwrite this method to draw an icon of the minized view. If 'view' is NULL this method does nothing.
