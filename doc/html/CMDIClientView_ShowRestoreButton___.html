CMDIClientView::ShowRestoreButton
Displays the restore button
virtual void ShowRestoreButton();


Remarks
Displays the restore button right of the key-menubar of the window. This method does nothing, it the restore button is already displayed. The restore button sends a MDI_MSG_RESTORE_VIEW to this view when its activated.
