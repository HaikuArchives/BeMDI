CMDIClientView::TileViews
Tiles the views either horizontal or vertical.
virtual void TileViews(bool horz);


Remarks
This method is called when a MDI_MSG_TILE_VIEWS message is received, but it can also be called directly. This method has no effect, if its called when some view is maximized.
