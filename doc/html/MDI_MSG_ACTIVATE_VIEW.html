  Globals and Constants

MDI_MSG_ACTIVATE_VIEW
Message id.
const int32 MDI_MSG_ACTIVATE_VIEW;


Remarks
A message with this 'what' field can be send to a CMDIClientView to activate a MDI view. Which MDI view is activated is specified in the MDI_MSG_DATA_VIEW data field of the message.
