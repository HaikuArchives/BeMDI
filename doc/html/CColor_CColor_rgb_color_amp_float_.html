  CColor

CColor::CColor
Construct a CColor object from color and "tint level".
CColor(const rgb_color &other, float tint);


Remarks

Fills a CColor object with a darker or lighter copy of 'other' depending on 'tint'.

Possible values for 'tint':

B_LIGHTEN_MAX_TINT
B_LIGHTEN_2_TINT
B_LIGHTEN_1_TINT
B_NO_TINT
B_DARKEN_1_TINT
B_DARKEN_2_TINT
B_DARKEN_3_TINT
B_DARKEN_4_TINT
B_DARKEN_MAX_TINT
