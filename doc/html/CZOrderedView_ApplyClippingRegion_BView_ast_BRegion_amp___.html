CZOrderedView::ApplyClippingRegion
Applies a new clipping region.
virtual void ApplyClippingRegion(BView *view, BRegion &region);


Remarks
This method takes the passed region and applies it to this view and all its children. To start this recursion you must set 'view' to NULL. The passed clipping region must be in screen coordinates.
