CZOrderedViewManager::cachedOrigin
Position on screen at last update of the cached screen rects.
BPoint cachedOrigin;


Remarks
If the postion of this view or any of its parents changes, the chached screen rects of the CZOrderedView objects get invalid. Therefore Z_UpdateCachedRects uses this member to determine if this view was moved.
