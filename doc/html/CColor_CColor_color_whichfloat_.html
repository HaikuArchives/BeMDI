  CColor

CColor::CColor
Construct a CColor object from default UI color.
CColor(color_which uiColor, float tint=B_NO_TINT);


Remarks

Fills a CColor object with a default UI color. Uses the BeOS functions ui_color and tint_color.

Possible values for 'tint':

B_LIGHTEN_MAX_TINT
B_LIGHTEN_2_TINT
B_LIGHTEN_1_TINT
B_NO_TINT
B_DARKEN_1_TINT
B_DARKEN_2_TINT
B_DARKEN_3_TINT
B_DARKEN_4_TINT
B_DARKEN_MAX_TINT
